Welcome to Oxagen! This guide will walk you through the basics so you can get up and running quickly.

## Prerequisites

Before you begin, make sure you have the following:

- An Oxagen account
- Node.js 18 or later installed
- A terminal or command-line interface

## Quick Start

### 1. Install the CLI

```bash
npm install -g oxagen
```

### 2. Authenticate

```bash
oxagen auth login
```

### 3. Initialize a Project

```bash
oxagen init my-project
cd my-project
```

### 4. Run Your First Command

```bash
oxagen run
```
